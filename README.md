# Persian sans font family

> Under MIT license persian and arbic font

# Usage

```bash
npm i persiansans
```


link css:

```html
<link rel="stylesheet" href="node_modules/persiansans/fontface.css">
```

use in css:

```css
*{
	font-family: PersianSans, sans-serif;
}
```
